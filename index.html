Goal Of The Game

In Tetris your goal is to fill up a row all the way across to get points.
The catch is that the more lines that you get the faster the blocks fall.
Get more lines with one block to get more points.




Basic Controls

Move Left: Left Arrow Key

Move Right: Right Arrow Key

Turn Counter Clockwise: Up Arrow Key

Turn Clockwise: Down Arrow Key

Drop a block: Space Bar (make sure to have clicked off of the new game button)

To make a block drop faster: hold Shift




Scoring

1 line  : 100 Points

2 lines : 300 Points

3 lines : 600 Points

4 lines : 1200 Points
